Criteria

  1. Owner of the site must be a member of Agora Road's Macintosh Cafe
  2. Site must have at least one piece of content (i.e. No blank pages with links to vestigial social media).
  3. Applicants must submit an 88x31 button.
  4. Members must place the webring on their site. See below

Webring widget

You can additionally link and highlight a specific user on the webring via the html ID, i.e. for highlighting risingthumb.

An API has been kindly provided by Nobo that allows linking to next and previous members of the webring using this ID. The format is 

https://agoraroad.goeshard.org/ID/next
https://agoraroad.goeshard.org/ID/prev
where you replace ID with your ID. As an example for RisingThumb's next and previous links.

Feel free to include links to your webring neighbours.

How can I join?

  • Message RisingThumb on the AgoraRoad forum with your (1) Username, (2) Website URL, (3) Link to your 88x31 Button, (4) OPTIONAL: RSS Link
  • Alternatively email RisingThumb at the address risingthumb (at) risingthumb (dot) xyz
  • Alternatively create a pull request adding this in on the Github page for the webring
  • Alternatively create a github issue with the above details requesting to join.

RSS notes

For websites that have them, a link to their RSS feed has been provided. Many of these RSS feeds are aggregated in the Agora Road Feeds website provided by Zinricky. If your RSS feed is missing, feel free to message a maintainer about this!

Found a broken link? Broken buttom image? Broken RSS link? Please report it or message a maintainer about it.
